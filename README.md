## 🚀 Sobre mim

Sou estudante de **Engenharia de Controle e Automação** e estou em constante evolução na área de **Tecnologia da Informação**, com foco em desenvolvimento de sistemas, automações, banco de dados e soluções que transformam processos manuais em ferramentas mais organizadas, rápidas e eficientes.

Tenho interesse em criar aplicações que resolvam problemas reais, principalmente envolvendo **backend**, **APIs**, **PostgreSQL**, **Docker**, **ETL**, **dashboards**, **automações**, **integrações entre sistemas** e soluções utilizando **inteligência artificial**.

Atualmente estou construindo minha base técnica, desenvolvendo projetos práticos e organizando meu GitHub como um portfólio profissional para registrar minha evolução.
